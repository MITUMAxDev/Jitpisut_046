Welcome! 
          Lua เป็นภาษาการเขียนโปรแกรมที่ทรงพลังและรวดเร็ว ซึ่งง่ายต่อการเรียนรู้และใช้งาน และยังสามารถฝังลงในแอปพลิเคชันของคุณได้อย่างง่ายดาย
Lua ถูกออกแบบมาให้เป็นภาษาสคริปต์ที่มีน้ำหนักเบาและสามารถฝังตัวได้ จึงถูกนำมาใช้ในแอปพลิเคชันหลากหลายประเภท ตั้งแต่เกมจนถึงเว็บแอปพลิเคชันและการประมวลผลภาพ
คุณสามารถดูรายละเอียดเพิ่มเติมเกี่ยวกับ Lua ได้บนหน้าอื่นๆของ เว็บไซต์
Building from source 
          Lua นั้นติดตั้งได้ง่ายมาก เพียงแค่ดาวน์โหลดและทำตามคำแนะนำในแพ็กเกจ
นี่คือตัวอย่างการใช้งานเทอร์มินัลที่เรียบง่ายในการดาวน์โหลดเวอร์ชันล่าสุดของ Lua และทำการสร้างในระบบ Linux:
        
    curl -L -R -O https://www.lua.org/ftp/lua-5.4.7.tar.gz
    tar zxf lua-5.4.7.tar.gz
    cd lua-5.4.7
    make all test

หากคุณใช้ Windows และต้องการ Build Lua มีคำแนะนำโดยละเอียดใน
